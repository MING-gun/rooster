# "小公鸡" APP

一款面向大学生的合同智能审查APP

# 项目介绍

📱 项目简介

合同智能审查App是一款专为大学生群体设计的智能法律辅助工具，旨在帮助大学生在租房、兼职、实习等常见场景中，快速识别合同中的潜在风险条款，提供通俗易懂的法律解读和修改建议，避免因法律知识不足而遭受权益侵害。


✨ 核心功能

🔍 合同风险扫描

一键检测合同中隐藏的“霸王条款”与潜在陷阱，重点提示违约金、责任不对等、模糊解释等常见风险点，助你快速把握关键问题。

👨‍⚖️ 即时律师答疑

提供线上一对一专业律师咨询服务，针对你合同中的具体条款进行针对性解读与修改建议，关键问题不必独自困惑。

💬 互助社区

创建同龄人交流空间，可以分享签约经历、吐槽条款陷阱、互助解答疑问，从真实故事中积累经验，实现朋辈支持。

🎓 法律知识

定期推出由专业律师主讲的合同知识微课、签约技巧直播与普法专栏，帮你逐步建立法律风险防范意识，成长为自己的“第一法务”。
